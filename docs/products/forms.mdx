## Counter input

A Counter is a Form Element containing a label and a number Input that can be increased or decreased using companion Button Icons.

## Segmented controls

A Segment Control is a horizontal control made of multiple segments. It provides a compact means of grouping together a number of controls.

## Inline forms
Use the .form-inline className to display a series of labels, form controls, and buttons on a single horizontal row. Form controls within inline forms vary slightly from their default states.
<Playground>
<form className="form-inline">
  <label>Name</label>

</form>
</Playground>
